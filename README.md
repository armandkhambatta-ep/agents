\# Create Your First Agent

A companion activity for ExperiencePoint's Work Better with AI 201 workshop.

It walks you through building your first AI agent in about four minutes, at your own pace. You pick the tool you use (Copilot, ChatGPT, or Claude), and the page guides you step by step with screenshots. If you don't have an agent builder, there's a fourth path that teaches the same thinking using any AI chat.
